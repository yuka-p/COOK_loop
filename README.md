## ✨サービス概要
毎日のごはんづくりは家族のために欠かせない一方で、「何を作ろう？」と考えることが大きな負担になることがあります。
COOK loopはそんな日々の献立を考えるストレスを減らし、効率よく手作りを続けたい気持ちに寄り添う献立提案サポートアプリです。

## 💭このサービスへの思い・作りたい理由
私は料理が好きではありません。
正確には、やるべきことに追われる中で時間までに必ずごはんを用意しなければならない、というプレッシャーに疲れてしまっているのだと思います。
家族の健康を支える食事は大切ですが、毎日の献立を考えるのは大変です。栄養バランス、家族の好み、冷蔵庫の食材、調理にかけられる時間など考えることが多く、他のことをしていても頭の片隅でいつも「今日のごはん何にしよう？」がぐるぐるしています。

- 外食や市販の惣菜は味が濃く、栄養も偏りがち
- レシピ検索サイトは情報が多すぎて探すだけで疲れてしまう
- AI献立は普段使わない食材も多く、余った食材の活用方法をまた考えなければならない
- 新しいレシピで作る場合、料理中で手が汚れているなかでのスマホ操作が大変
- 苦労してせっかく作った料理も、慣れない食材や味付けだと結局子供が食べてくれない

このような経験、みなさんもあるのではないでしょうか。
私自身、家族の健康や節約のために手作りを続けたいけれど、もう考えたくないと感じることが何度もありました。
自分が持っている定番のメニューを効率よく回すことで、考える手間を最小限に料理ができる仕組みを作りたい。そんな発想から生まれたアプリです。

## 🪄サービスの利用イメージ
1. マイメニューの登録
ユーザーが普段よく作っている料理を登録します。定番メニューの一覧からメニューを選択する方法と、ユーザーが手入力する2つの登録方法があります。本人がわかればいいので、料理手順や食材の登録は不要です。
2. 献立提案機能
登録されたメニューのなかで直近で作っていないものを優先して表示します。
3. 献立リストの作成
気に入ったメニューを選び、献立リストを作成します。
4. 実績の記録
実際に作ったかどうかを記録し、次回以降の提案に反映します。
↓
いつも作っているメニューなので料理中にレシピの確認が不要。
料理手順も慣れている作業なので時間短縮になる。
作っても子供が食べないというリスクが少ない。
お弁当メニューや冷凍OKなど任意のタグをつけて絞り込みも可能。
自分のもっているレシピを効率よく回すことで今日何作ろうの悩みを減らし、日々の料理の負担を軽減することを目指します。

## 🪄サービスの差別化ポイント・推しポイント
国内の主要な献立・レシピアプリは、「新しいレシピを探す・学ぶ」体験を中心に設計されています。豊富なレシピ検索、動画解説、AIによる提案機能などが強みですが、どれも新しい料理との出会いに重きを置いています。
一方で日々の家庭料理では都度新しいレシピを探すよりも、自分の定番メニューを効率よく回すことのほうが現実的です。しかしそのニーズに応えるアプリは国内ではほとんど存在していません。
COOK loopはユーザー自身の定番メニューを登録・管理し、「最近作っていない料理」を優先的に提案することで、考える負担を減らしながら手作りを続けられることを目指した献立サポートアプリです。
